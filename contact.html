La la land

Porjeto desenvolvido por alunos somente para fins educacionais, esse site tem como objetivo aplicar os conceitos de html e css vistos no módulo 8 do do programa Top Coder's, desenvolvido entre a parceria Ada e Bando Safra para formar desenvolvedores Web Full-Stack. As fotos e acomodações desse site se referem a lugares reais, contudo todos os dados de reserva e contato são ficticios.
